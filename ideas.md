# Keturah Stone Enhancement Website - Design Brainstorm

## Project Context
An interactive showcase of luxury marble transformations for the Keturah Stone Enhancement project. The website will feature before/after comparisons of Travertine to Bardiglio and Venatino marble across 25+ luxury interior spaces.

## Idea 1: Brutalist Gallery Noir

**Design Movement:** Neo-Brutalism meets High-End Editorial

**Core Principles:**
- Raw, unapologetic typography with extreme weight contrasts
- Stark black backgrounds with marble textures bleeding through
- Asymmetric grid layouts that break conventional alignment
- Dramatic negative space as a compositional tool

**Color Philosophy:**
- Primary: Pure black (#000000) as the void canvas
- Secondary: Off-white (#F5F5F0) for stark contrast
- Accent: Warm gold (#C9A962) for luxury touches
- Marble gray tones (#8A8A8A, #B8B8B8) echoing the stone itself

**Layout Paradigm:**
- Offset grid with images breaking container boundaries
- Full-bleed hero sections with text overlays
- Staggered card layouts with varying heights
- Horizontal scroll sections for comparison galleries

**Signature Elements:**
- Thick black borders around comparison frames
- Oversized serif numerals as section markers
- Grain texture overlay on all surfaces

**Interaction Philosophy:**
- Aggressive hover states with scale and shadow shifts
- Comparison sliders with chunky, visible handles
- Scroll-triggered reveals with hard cuts, not fades

**Animation:**
- Abrupt entrance animations (clip-path reveals)
- Parallax depth on marble images
- Text scramble effects on hover

**Typography System:**
- Display: "Bebas Neue" or "Oswald" (900 weight) for headlines
- Body: "Space Grotesk" (400) for descriptions
- Accent: Monospace for technical specs
</text>

## Idea 2: Liquid Marble Immersion

**Design Movement:** Organic Fluidity meets Luxury Minimalism

**Core Principles:**
- Flowing, organic shapes that mimic marble veining
- Soft gradients transitioning between marble tones
- Generous whitespace with floating elements
- Seamless visual flow guiding the eye through content

**Color Philosophy:**
- Primary: Warm cream (#FAF8F5) as the base canvas
- Secondary: Deep charcoal (#2A2A2A) for text hierarchy

- Highlight: Muted terracotta (#C4A484) for warmth

**Layout Paradigm:**
- Curved section dividers mimicking marble veins
- Overlapping image frames with soft shadows
- Centered hero with radial content expansion
- Floating cards with subtle rotation

**Signature Elements:**
- SVG wave dividers between sections
- Soft blob shapes as background accents
- Circular image masks for detail shots

**Interaction Philosophy:**
- Buttery smooth transitions (300-400ms ease-out)
- Magnetic cursor effects near interactive elements
- Gentle parallax creating depth layers

**Animation:**
- Smooth fade-up entrances with stagger
- Image zoom on scroll within masked containers
- Floating ambient motion on decorative elements

**Typography System:**
- Display: "Cormorant Garamond" (600) for elegant headlines

## Selected Approach: **Idea 2 - Liquid Marble Immersion**

This approach best reflects the luxury and organic beauty of marble while providing an immersive, gallery-like experience. The flowing aesthetic mirrors the natural veining patterns of both Bardiglio and Venatino marble, creating a cohesive visual language that celebrates the material itself.

### Implementation Notes:
- Use curved SVG dividers between sections
- Implement smooth comparison sliders with soft shadows
- Create floating card layouts for the gallery grid
- Apply subtle parallax effects on marble images
- Use cream/warm white backgrounds with charcoal text
- Incorporate soft blue-gray accents reflecting Venatino tones
